## Installation

Install Witchery with npm
```bash
  git clone https://github.com/DrunkTaric/Witchery
  cd Witchery
  npm install
```

Install Witchery with pnpm
```bash
  git clone https://github.com/DrunkTaric/Witchery
  cd Witchery
  pnpm install
```
## Deployment

To deploy this on your local machine 

1. Make production build
```bash
    npm run build
```
2. Start the server
```bash
    npm run start
```
